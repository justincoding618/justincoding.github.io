ESMFold Protein Structure Predictor

Back around 2022, the Data Professor created an ESMFold application that would predict a protein structure from a given sequence. My project builds off of the Data Professor's ESMFold application. However, I relied on Claude AI to help optimize the code.

The main application has four defined functions:

  • The validate_sequence() function takes one argument and tries to read a given protein sequence.
  • The render_molecule() function takes three arguments. The first is a pdbfile of the protein. The second is the color scheme to visualize the protein. The third is to enable spinning so that the protein structure can be viewed on different sides.
  • The plot_plddt_scores() function takes one argument, which is the structure of the protein. It produces a figure that shows the plot of each residue position and the respective pLDDT score. A score that is higher than 90 indicates a very high confidence score, while a score less than 50 indicates a low confidence score.
  • The update() function would take a given sequence string and try to update the current protein structure to a given protein sequence.

The ESMFold application uses a Web API using the URL https://api.esmatlas.com/foldSequence/v1/pdb/. If that URL is not available, then users can use the Google Colab notebook to test a given protein sequence. They can then upload the pdb results to test the application.

Currently, this application can be tested locally on your machine. You need to first create a virtual environment through Anaconda and then activate the virtual enviornment. You can learn more how to do this thorugh the README page of the repo.
